## üéØ Project Overview

**DragonTrade Agent (ALGOM)** is an AI-powered crypto trading education and paper trading bot with:

- **Educational Content**: Comprehensive trading education
- **Paper Trading**: Simulated trading with real market data (Bybit & Binance)
- **Social Integration**: Twitter bot with automated posting
- **Marketing Automation**: CMO AIPA integration for brand building
- **Multi-Service Architecture**: Distributed system with PostgreSQL backend

### Technology Stack
- **AI Framework**: Eliza OS (@elizaos/core)
- **AI Models**: Anthropic Claude, OpenAI, Groq
- **Social Platform**: Twitter API v2
- **Market Data**: CoinGecko, CoinMarketCap, CCXT
- **Deployment**: Railway (multi-process)
- **Database**: PostgreSQL

## üîÑ Recent Major Changes

1. **CMO AIPA Integration** - 20-post cycle with balanced content (30% paper trading, 30% marketing, 40% education)
2. **PostgreSQL Integration** - Multi-service data sharing for automatic reporting

4. **Dual Exchange Paper Trading** - Bybit and Binance comparison testing
